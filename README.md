Find standings for major soccer/football leagues directly from your terminal.

#Modules to Install:
Open Mac terminal and just write the 'import statement' below
  - Pip
```
$sudo easy_install pip
```
  - Termcolor
```
$pip install termcolor
```
  - Lxml
```
$pip install lxml
```
